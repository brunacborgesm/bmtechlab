# Studio Components

This directory contains the UI components specifically designed for the BM Tech Lab studio.

### Structure:
- **Navigation**: Custom landing page navigation.
- **Sections**: Self-contained blocks like Hero, Services, Portfolio, FAQ, and CTA.
- **Interactive Elements**: Specialized animations and transitions using Framer Motion and GSAP.

Each component is built to be modular, ensuring that the studio's identity is consistent across all pages.
